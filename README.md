# market-api
Session cart with filtering route

Выполните команду composer
> composer install

Выполните команду npm
> npm install

Распакуйте файл .env и введите данные

Выполните команду миграции
> php artisan migrate

Заполните бд командой
> php artisan db:seed

Запустите сервер laravel
> php artisan serve

Запустите npm
npm run dev
